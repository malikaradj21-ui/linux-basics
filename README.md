"# Linux Basics (Lernprojekt)" 
"" 
"Dies ist ein Schritt-für-Schritt Lernprojekt für Linux für absolute Anfänger." 
"Ziel ist es, die Grundlagen von Linux zu lernen: Terminal, Dateien, Berechtigungen und einfache Skripte." 
"" 
"Struktur des Projekts:" 
"- lessons/: Enthält die Lernlektionen (Markdown-Dateien)." 
"- exercises/: Enthält Übungsdateien und kleine Skripte." 
"- assets/: Optional für Bilder oder Screenshots." 
"" 
"Jeder Schritt wird als separater Commit auf GitHub hochgeladen, um den Fortschritt zu dokumentieren." 
"" 
"Wichtige Hinweise:" 
"- Keine Passwörter oder sensible Daten ins Repository hochladen!" 
"- Alle Übungen sind klein, damit das Projekt wenig Speicherplatz benötigt."
