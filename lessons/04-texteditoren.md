# Lektion 04 – Texteditoren in Linux

In dieser Lektion lernst du die grundlegenden Texteditoren in Linux kennen und wie man Dateien bearbeitet.

Wichtige Punkte:
- nano: einfacher Terminal-Editor (nicht immer verfügbar auf Windows)
- vim: leistungsfähiger Terminal-Editor (nicht immer verfügbar auf Windows)
- echo "Text" > datei.txt: erstellt eine Datei mit Inhalt
- echo "Weitere Zeile" >> datei.txt: fügt eine neue Zeile hinzu

Erklärungen zu den Befehlen:
- echo "Text" > datei.txt: erstellt eine Datei mit Inhalt
- echo "Weitere Zeile" >> datei.txt: fügt eine neue Zeile hinzu
- cat datei.txt: zeigt den Inhalt einer Datei
- nano datei.txt: öffnet die Datei im Terminal-Editor (Windows evtl. nicht verfügbar)
- vim datei.txt: öffnet die Datei im Terminal-Editor (Windows evtl. nicht verfügbar)
