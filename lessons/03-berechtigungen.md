# Lektion 03 – Benutzerrechte und Berechtigungen

In dieser Lektion lernst du die grundlegenden Berechtigungen von Dateien und Ordnern kennen.

Wichtige Befehle:
- ls -l: zeigt die Dateirechte
- chmod: ändert die Berechtigungen (eingeschränkt auf Windows)
- chown: ändert den Besitzer (nicht voll funktionsfähig auf Windows)
- echo "" > [Dateiname]: neue leere Datei erstellen
- mkdir [Ordnername]: neuen Ordner erstellen
