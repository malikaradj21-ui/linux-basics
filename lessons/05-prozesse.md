# Lektion 05 – Arbeiten mit Prozessen

In dieser Lektion lernst du, wie man Prozesse in Linux/Windows über Git Bash überwacht und verwaltet.

Wichtige Befehle:
- ps: zeigt die laufenden Prozesse
- ps aux: detaillierte Prozessliste
- kill [PID]: beendet einen Prozess (funktioniert teilweise auf Windows)
- jobs: zeigt Hintergrundprozesse

Erklärungen zu den Befehlen:
- ps: zeigt die laufenden Prozesse
- ps aux: detaillierte Liste aller Prozesse
- kill [PID]: beendet einen Prozess
- jobs: zeigt die Hintergrundprozesse
- & am Ende eines Befehls: startet den Prozess im Hintergrund
